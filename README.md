# Suraj - Developer Portfolio

A modern, responsive React.js developer portfolio showcasing Suraj's skills, projects, and experience.

## Features

- **Modern Tech Stack**: Built with React.js, TypeScript, TailwindCSS
- **Smooth Animations**: Framer Motion for page transitions and micro-interactions
- **Scroll Animations**: AOS.js for reveal-on-scroll effects
- **Typewriter Effect**: Typed.js for dynamic role display
- **Dark/Light Mode**: Theme switcher with localStorage persistence
- **Fully Responsive**: Mobile-first design that works on all devices
- **SEO Optimized**: Meta tags and semantic HTML for better search visibility

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Framer Motion
- AOS.js
- Typed.js
- FontAwesome Icons

## Sections

1. **Hero** - Eye-catching introduction with animated typewriter effect
2. **About** - Personal background and contact details
3. **Skills** - Technical skills with animated progress bars
4. **Education** - Academic background and certifications
5. **Projects** - Portfolio of completed projects with live demos
6. **Testimonials** - Client feedback and reviews
7. **Contact** - Contact form and social links
8. **Footer** - Quick links and social media

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
